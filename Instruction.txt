FrontEnd:

საჭიროა ლინუქსის ოპერაციული სისტემა, მაგალითად: ubuntu 22.04

ასევე ios სმარტფონზე დააყენეთ აპლიკაცია ExpoGo


1. დაკლონეთ კოდის რეპოზიტორია(ან გამოიყენეთ ლოკალური ფოლდერები):
   git pull (link)

2. დააყენეთ npm პაკეტი:
   apt install npm

3. დააყენეთ expo-cli:
   npm install -g expo-cli

4. npx expo login (თუ არ ხართ რეგისტრირებული, რეგისტრაცია გაიარეთ აქ: https://expo.dev/signup)

5. შექმენით .env ფაილი,რომელიც შეიცავს ბექენდის ენდფოინთს:
   echo "EXPO_PUBLIC_API_BASE_URL=http://localhost:3000" > .env

6. დასტარტეთ აპლიკაცია:
   npx expo start --tunnel --go

დაასკანერეთ QR კოდი რომელსაც გამოიტანს კონსოლში და გაგიხსნით აპლიკაციაში: Expo Go

BackEnd:

საჭიროა ლინუქსის ოპერაციული სისტემა, მაგალითად: ubuntu 22.04

1. დაკლონეთ კოდის რეპოზიტორია (ან გამოიყენეთ ლოკალური ფოლდერები):
   git pull (link)

2. დააყენეთ საჭირო პაკეტები:
   docker.io და docker-compose

3. ჩადექით დაკლონილ პროექტში და დასტარტეთ აპლიკაცია:
   docker-compose --env-file .env up -d
